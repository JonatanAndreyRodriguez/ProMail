# Remove-AuthorizedSender

## SYNOPSIS
Elimina registros de remitentes autorizados.

## SYNTAX

```
Remove-AuthorizedSender [-EmailAdressObject] <Object>
```

## DESCRIPTION
Elimina registros de remitentes autorizados.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Get-AuthorizedSender | Remove-AuthorizedSender
```
