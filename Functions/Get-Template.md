# Get-Template

## SYNOPSIS
Obtiene la información de la plantilla de una regla.

## SYNTAX

```
Get-Template [-IdRule] <Int32>
```

## DESCRIPTION
Obtiene la información de la plantilla de una regla.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Get-Template -IdRule 1
```
