# Remove-Subject

## SYNOPSIS
Elimina registros de asuntos válidos para los mensajes de correo entrantes.

## SYNTAX

```
Remove-Subject [-SubjectObject] <Object>
```

## DESCRIPTION
Elimina registros de asuntos válidos para los mensajes de correo entrantes.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Get-Subject | Remove-Subject
```
