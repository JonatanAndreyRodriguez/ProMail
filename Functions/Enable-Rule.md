# Enable-Rule

## SYNOPSIS
Habilita las reglas a ser monitoreadas.

## SYNTAX

### WithIdRule
```
Enable-Rule -IdRule <Int32>
```

### WithObject
```
Enable-Rule -ObjectRule <Object>
```

## DESCRIPTION
Habilita las reglas a ser monitoreadas.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```powershell
Get-Rule | Enable-Rule
```

### -------------------------- EXAMPLE 2 --------------------------
```powershell
Enable-Rule -IdRule 2
```
